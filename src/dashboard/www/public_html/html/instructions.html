How to use this dashboard

Navigate the dashboard to explore research impact case studies submitted by academic institutions in the United Kingdom to the 2021 Research Excellence Framework.

Click on elements of the dashboard to filter results to Impact Case Studies that most interest you:

  • Select a topic from the left panel to filter results to research in a topic area.
  • Gain an understanding of the topic by reading the narrative with examples of impact case studies in this research area.
  • Filter results by funder by clicking on bars in the funder barplot
  • Filter results by the postcode of UK academic institutions by clicking the top-left map
  • Filter results by country of research beneficiaries by clicking the bottom-right map
  • Filter results by units of assessment by selecting items from the drop-down list in the bottom-left panel
  • View and download our Enhanced REF Database for the filtered set of Impact Case Studies by clicking “View and Download Data: Enhanced REF Database” in the bottom-left of the dashboard
  • Explore advanced options by clicking the menu icon in the top-right of the dashboard

Application Programming Interface (API)

Researchers and web developers can use our API to query the Enhanced REF database underlying the dashboard. Documentation for the API can be found at https://shape-impact.co.uk/api/ui.
